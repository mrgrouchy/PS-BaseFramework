# Security Policy

## Supported Versions

Security updates are applied to the latest version on `master`.

| Version | Supported |
| ------- | --------- |
| master  | Yes       |
| older   | No        |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report privately with:
- clear description of the issue
- reproduction steps or proof of concept
- impact assessment
- suggested mitigation if available

If private reporting is not yet configured for this repository, contact the maintainer directly and mark the message as `SECURITY`.
